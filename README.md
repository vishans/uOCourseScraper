# uoScrape
A simple uOttawa course scraper written in Python.
All the data scraped is publicly available at https://catalogue.uottawa.ca/en/courses/.

# What is scraped exactly?
+ subject_code
+ course_number
+ name (course name)
+ prerequisites
+ units 

*Note: All the attributes are of type ```str```*
